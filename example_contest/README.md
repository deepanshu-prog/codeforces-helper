# Example Contest

This directory is a sample output generated by `cf fetch 1234` for demonstration purposes. It contains scraped test cases from [Codeforces Round #590 (Div. 3)](https://codeforces.com/contest/1234).

## Directory Structure

Each problem folder contains:
- `input*.txt` — Sample input for the problem
- `output*.txt` — Expected output for the problem
- `solution.cpp` — Solution file (created from template)

After running tests, these are also created:
- `out*.txt` — Output generated by your solution
- `result.txt` — Summary of test results (pass/fail)

## Try It Out

```bash
# Run tests for problem D
cf test example_contest/D

# Or problems E, F which have multiple test cases
cf test example_contest/E
cf test example_contest/F
```
